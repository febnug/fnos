A very <b><code>dumb</code></b> simple minimalist operating system, written from scratch using `x86` assembly language (`8086` flavour) real mode.

Soon

<h3>How to build</h3>

```
make
```

<h3>How to run</h3>

Using <a href="https://github.com/qemu/qemu">QEMU</a> :

```
qemu-system-i386 -drive format=raw,file=./fnos-beta.img
```
